Mailing List
The mailing list allows us to inform all subscribers about the release of the latest version of Servant Salamander. Do not hesitate to join now, it's the easiest way to get information about the release of new versions. We assure you that we won't use your e-mail address for any other purpose.

Beta Testing Program
The beta versions of Servant Salamander are very similar to the final versions with respect to safety and stability. The biggest advantage of the beta version is being up to date with the most recent working versions of Servant Salamander. We release the beta versions without changes in documentation and other backstage works, so that's easy for us.

Users List
We would be happy if you would send us a message about how you use Servant Salamander. We are very interested in the number of Servant Salamander users. Please add yourself to our users list by sending us your name. You can also recommend a language which Servant Salamander should be written in. This will help us to decide whether to make that language version or not.

To send us information use the following form. You needn't fill in all fields. Enter only information which we'll need for selected services. For example, we always need your name, but we need your e-mail only if you want to subcribe to the mailing list or to the beta testing list.
